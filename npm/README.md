## Prerequisites

- Java 11 or higher
- Maven 3.6+
- Chrome/Firefox/Edge browser installed

## Running Tests

### Run all tests:
```bash
mvn clean test
```

### Run with specific browser:
```bash
mvn clean test -Dbrowser=chrome
mvn clean test -Dbrowser=firefox
mvn clean test -Dbrowser=edge
```

### Run specific test class:
```bash
mvn test -Dtest=EmployeeValidationTest
mvn test -Dtest=EmployeeXSSSecurityTest
```

## Test Reports

TestNG HTML reports are generated in `target/surefire-reports/index.html`

## Page Object Model (POM)

The framework follows Page Object Model design pattern:
- **LoginPage**: Handles login functionality
- **PIMPage**: Handles employee management operations
- **BaseTest**: Manages WebDriver lifecycle

## Framework Features

- Page Object Model architecture
- TestNG for test execution and reporting
- Selenium WebDriver 4.x
- Explicit waits for reliable element interactions
- Maven for dependency management
- Configurable browser selection via system properties
